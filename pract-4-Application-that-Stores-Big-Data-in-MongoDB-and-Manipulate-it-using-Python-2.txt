1. Aim

2. Theory
What is MongoDB?
MongoDB is a NoSQL database.
It stores data in JSON-like documents.
It is used for handling Big Data.
MongoDB is flexible because rows can have different columns.

Example Document:

{
  "name": "Rahul",
  "age": 22
}

6. Python Program
from pymongo import MongoClient

# Connect MongoDB
client = MongoClient("mongodb+srv://username:password@cluster0.mongodb.net/")

# Create Database
db = client["studentDB"]

# Create Collection
collection = db["students"]

# Insert Data
data = {
    "name": "Rahul",
    "age": 22,
    "course": "BCA"
}

collection.insert_one(data)

print("Data Inserted Successfully")

# Display Data
for record in collection.find():
    print(record)
