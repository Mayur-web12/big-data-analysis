Write a Program to Illustrate the Working of Jaql
1. Aim
To illustrate the working of Jaql operations such as filtering, transformation, grouping, and aggregation using Python.

2. Theory
Jaql is a query language used for processing and analyzing JSON data in Big Data applications.
It is mainly used with Hadoop for handling semi-structured data.

(Write only 2–3 lines in exam)

3. Requirements
1. Python
2. JSON Module
3. Any Python IDE

4. Program Code
import json

# Step 1: Sample JSON data

data = [
    {"name": "Alice", "age": 25, "city": "Mumbai"},
    {"name": "Bob", "age": 30, "city": "Delhi"},
    {"name": "Charlie", "age": 35, "city": "Mumbai"}
]

print("Original Data:")
print(json.dumps(data, indent=2))

# Step 2: Filter (city = Mumbai)

filtered = [p for p in data if p["city"] == "Mumbai"]

print("\nFiltered Data (city = Mumbai):")
print(json.dumps(filtered, indent=2))

# Step 3: Transform (get only names)

names = [p["name"] for p in filtered]

print("\nNames from filtered data:")
print(names)

# Step 4: Group by city

grouped = {}

for p in data:
    city = p["city"]

    if city not in grouped:
        grouped[city] = []

    grouped[city].append(p)

print("\nGrouped Data by City:")
print(json.dumps(grouped, indent=2))

# Step 5: Aggregate (average age in Mumbai)

avg_age = sum(p["age"] for p in filtered) / len(filtered)

print("\nAverage Age in Mumbai:")
print(avg_age)
