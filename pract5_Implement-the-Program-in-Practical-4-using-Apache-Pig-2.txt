1. Aim

To perform Big Data operations such as grouping, sorting, joining, and filtering using Apache Pig.

2. Theory
What is Apache Pig?
Apache Pig is a platform used for analyzing large datasets.
It uses a scripting language called Pig Latin.
Pig works on Hadoop and HDFS.
It simplifies Big Data processing.

10. Joining Operation
Load Customer Data
customers1 = LOAD 'customers1.txt'
USING PigStorage(',')
AS (id:int,name:chararray,age:int,address:chararray,salary:int);

customers2 = LOAD 'customers2.txt'
USING PigStorage(',')
AS (id:int,name:chararray,age:int,address:chararray,salary:int);
Join Code
customers3 = JOIN customers1 BY id, customers2 BY id;
DUMP customers3;
Explanation
JOIN combines matching records from two datasets.
Output
(1,Ramesh,32,Ahmedabad,2000,1,Ramesh,32,Ahmedabad,2000)

(2,Khilan,25,Delhi,1500,2,Khilan,25,Delhi,1500)

(3,Kaushik,23,Kota,2000,3,Kaushik,23,Kota,2000)
