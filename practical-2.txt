Implement Word Count / Frequency Program using MapReduce

1. Aim

To count frequency of words using MapReduce in R.

2. Program
# Input text
text <- c(
  "Big data is Big",
  "Big data is powerful",
  "R is powerful"
)

# Convert to lowercase
text <- tolower(text)

# Remove punctuation
text <- gsub("[[:punct:]]", "", text)

# Split words
words <- unlist(strsplit(text, "\\s+"))

# Map phase
map_output <- data.frame(word = words, count = 1)

# Reduce phase
reduce_output <- aggregate(count ~ word,
                           data = map_output,
                           sum)

print(reduce_output)


3. Explanation
Step 1:

Input text is taken.

Step 2:

Convert text into lowercase using tolower().

Step 3:

Remove punctuation using gsub().

Step 4:

Split text into words using strsplit().

Step 5: Map Phase

Assign count 1 to each word.

Step 6: Reduce Phase

Add counts of same words using aggregate().

4. Output
Word	Count
big	3
data	2
is	3
powerful 2
r	1

5.Result
Word frequency was calculated successfully using MapReduce.

6. Conclusion
The MapReduce technique was successfully used to count word frequency.
